## ğŸ”‘ API Setup

PyWeather-QT uses the [OpenWeatherMap API](https://openweathermap.org/) to fetch weather data.

### How to get your API key:

1. Visit the [OpenWeatherMap website](https://openweathermap.org/)
2. Create a free account
3. Navigate to your profile and click on **"My API keys"**
4. Toggle on your API key using the toggle button
5. Copy your API key
6. In the app, click on the **'API key'** button
7. Paste your API key in the dialog box and click **'OK'**

> ğŸ’¡ **Note:** Your API key is stored locally on a .json file on your computer, the path of the json file is : **C:\Users\your username**, the name of the file is **.pyweatherqt_app_config.json**
